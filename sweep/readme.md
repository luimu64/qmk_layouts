# Aurora sweep

My personal layout based on mini-ryoku with heavy modifications like changing the alpha to dvorak and adding ä and ö.

### Base layer

```
     ,-------------------------------------------------.    ,-------------------------------------------------.
     | , .     | Ä       | Ö       | P       | Y       |    | F       | G       | C       | R       | L       |
     |-------------------------------------------------|    |-------------------------------------------------|
     | A       | O       | E       | U       | I       |    | D       | H       | T       | N       | S       |
     |-------------------------------------------------|    |-------------------------------------------------|
     | ' "     | Q       | J       | K       | X       |    | B       | M       | W       | V       | Z       |
     `-------------------------------------------------|    |-------------------------------------------------'
                                   | SPACE   | TAB     |    | ENTER   | BSPC    |
                                   `-------------------'    `-------------------'
```

NOTE: Holding tab and enter will switch to gaming layer.

### Symbol/Number/Arrow key layer

```
     ,-------------------------------------------------.    ,-------------------------------------------------.
     | ; :     | 7 &     | 8 *     | 9 (     | 0 )     |    | <       | >       | ▲       | CAPS    | PG UP   |
     |-------------------------------------------------|    |-------------------------------------------------|
     | / \     | 4 $     | 5 %     | 6 [     | ? ]     |    | # ^     | ◀       | ▼       | ▶     | PG DN   |
     |-------------------------------------------------|    |-------------------------------------------------|
     | ` ~     | 1 |     | 2 @     | 3 {     | ! }     |    | ESC     | HOME    | INSERT  | DEL     | END     |
     `-------------------------------------------------|    |-------------------------------------------------'
                                   | - _     |  = +    |    |         | WIN     |
                                   `-------------------'    `-------------------'

```

### Function/Mouse layer

```
     ,-------------------------------------------------.    ,-------------------------------------------------.
     | COPY    | F7      | F8      | F9      | F12     |    | ⇐      | ⇑       | ↑       | ⇓       | ⇒      |
     |-------------------------------------------------|    |-------------------------------------------------|
     | PASTE   | F4      | F5      | F6      | F11     |    | MIDDLE  | ←       | ↓       | →       | GAMING  |
     |-------------------------------------------------|    |-------------------------------------------------|
     | LED TOG | F1      | F2      | F3      | F10     |    | LED HUE | <<      | PLAY    | >>      | LED SAT |
     `-------------------------------------------------|    |-------------------------------------------------'
                                   | WIN     |         |    | RIGHT   | LEFT    |
                                   `-------------------'    `-------------------'
```

### Gaming layer

```
     ,-------------------------------------------------.    ,-------------------------------------------------.
     | TAB      | Q       | W       | E       | R      |    | T       | Y       | ▲       | I       | O       |
     |-------------------------------------------------|    |-------------------------------------------------|
     | SHIFT    | A       | S       | D       | F      |    | G       | ◀      | ▼       | ▶       | L       |
     |-------------------------------------------------|    |-------------------------------------------------|
     | CTRL     | Z       | X       | C       | V      |    | B       | N       | M       | U       | K       |
     `-------------------------------------------------|    |-------------------------------------------------'
                                   | SPACE   | ESC     |    | ENTER   | BSPC    |
                                   `-------------------'    `-------------------'
```
NOTE: Holding esc and enter will switch to base layer.
